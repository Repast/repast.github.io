Quick Start Guide

This Quick Start Guide is intended to provide a minimal set of instructions to download, install, and run Repast Simphony and provide the user with a path towards developing their own models. The Repast FAQ, Repast Downloads, and Repast Documentation pages all provide more details.

  1. Download and install Repast Simphony.
  2. Run Repast and import example models
  3. Select a Repast tutorial depending on your modeling needs and programming level:
    • If you are new to Repast or if your programming background is limited, start with the ReLogo Getting Started Guide. ReLogo is a computer language designed to help you quickly create simpler agent-based models.
    • After you are acquainted with Repast or if you know the Java computer language and you want to develop more complex Repast Simphony models, read the Repast Java Getting Started guide.
    • If you know the C++ computer language and you want to develop a large model to run on a supercomputer,work through the Repast HPC Web Tutorial or read the Repast HPC Manual. (If your knowledge of C++ is limited, to develop Repast HPC models you can also use an easier Logo-style C++ computer language, as described in Section 5 of the manual.)
