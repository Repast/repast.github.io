Repast Simphony External Analysis Tools

In addition to its advanced modeling facilities, Repast Simphony has integrated point-and-click connections to the following tools, which can be optionally downloaded and installed separately:

  • The R statistics environment,
  • The Weka data mining platform,
  • Many popular spreadsheets including Excel,
  • The Pajek network analysis tool.

Repast Simphony Platform Development

Developers who wish to modify the Repast Simphony platform should download Eclipse IDE for Eclipse Committers Oxygen. Please note that Repast Simphony development requires this feature-complete edition and will not work with the limited Eclipse Java Edition. Then use the Eclipse Update Manager to install the following components.

  • From the default Neon (note that Oxygen versions do not exist yet) update site (http://download.eclipse.org/releases/neon).
    • GMF Tooling 3.3.1
    • GMF Tooling Runtime Extensions 3.3.1
    • GMF Tooling Xtend2 Support
    • GMF Toolking SDK 3.3.1
    • Xpand SDK 2.2.0
  • From the Groovy Update Site (http://dist.springsource.org/release/GRECLIPSE/e4.7):
    • Extra Groovy Compilers
    • Groovy-Eclipse
    • Uncategorized
The repast repository can be cloned from the links provided at the Simphony GitHub repository. The master branch contains the code in a distribution, tagged with the distribution's version number, while all the work is done the development branch. Import all the projects into Eclipse. Note that you may need to Refresh (F5) and / or clean (Project -> Clean) some or all of the projects for an error-free compile.
