Repast Simphony External Analysis Tools

In addition to its advanced modeling facilities, Repast Simphony has integrated point-and-click connections to the following tools, which can be optionally downloaded and installed separately:

  • The R statistics environment,
  • The Weka data mining platform,
  • Many popular spreadsheets including Excel,
  • The Pajek network analysis tool.

Repast Simphony Platform Development

Developers who wish to modify the Repast Simphony platform should download Eclipse IDE for Eclipse Committers 4.4.2. Please note that the Repast Simphony update site requires this feature-complete edition and will not work with the limited Eclipse Java Edition. They should then use the Eclipse Update Manager to install the following components:

  • From the default Luna update site:
    • GMF Tooling 3.2.1
    • GMF Tooling Runtime Extensions 3.2.1
    • GMF Toolking SDK 3.2.1
    • Eclipse XML Editors and Tools. Depending on your OS, you may or may not see this as an option.
    • Xpand SDK
  • From the Groovy Update Site: http://dist.springsource.org/release/GRECLIPSE/e4.4
    • Extra Groovy Compilers
    • Groovy-Eclipse
The repast repository can be cloned with
git clone git://repast.git.sourceforge.net/gitroot/repast/repast.simphony
The mater branch contains the code in a distribution, while all the work is done the development branch.
