Package: repast.simphony.relogo.ide.wizards

[Java] Class NetlogoWizardPageOne

      • repast.simphony.relogo.ide.wizards.NetlogoWizardPageOne

      • The "New" wizard page allows setting the container for the new file as well as the file name. The page will only accept file name without the extension OR with the extension that matches the expected one (nlogo).
